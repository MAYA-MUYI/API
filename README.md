# movies_api

网络电影票购票平台综合信息接口(Movie Tickets Information API)  
实时查询国内电影购票平台场次，票价等信息。   
---

## V0.1
* 基于Python Django
* 使用scrapy获取三家信息
* 按城市-影院-场次-票价分层查询

## 1. 流程
>选择影片-> 查看详细 -> 选择影院 -> 选择位置 -> 确认支付 -> 返回订单
